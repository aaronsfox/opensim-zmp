OpenSim ZMP: exampleZmpTrackingSimulation
================================================

This example uses data from the publicly available dataset released with the paper:

Hamner SR & Delp SL (2013). Muscle contributions to fore-aft and vertical body mass center accelerations over a range of running speeds. J Biomech, 46(4): 780-787. doi: 10.1016/j.jbiomech.2012.11.024 

It uses a single gait cycle for one participant to demonstrate using the ZeroMomentPointGroundReactions component and associated classes in a tracking simulation.
